# Resume and Job Description Parser

This project is developed as part of Infosys Springboard – Python Full Stack Program (Milestone 1).

## Project Description
The application allows users to upload Resume and Job Description files
in TXT, PDF, or DOCX format. The system parses the documents, cleans the text,
and displays the content line by line using a colorful web interface.

## Technologies Used
- Python
- Flask
- HTML & CSS
- PyPDF2
- python-docx

## Features
- Upload Resume and Job Description
- Supports TXT, PDF, DOCX formats
- Line-by-line parsed output
- Colorful and user-friendly UI
- Runs on localhost

## How to Run the Project
1. Install dependencies:
   pip install flask PyPDF2 python-docx

2. Run the application:
   python app.py

3. Open browser:
   http://127.0.0.1:5000/

## Output
- Resume preview displayed on the left
- Job Description preview displayed on the right
- Parsed content shown line by line
